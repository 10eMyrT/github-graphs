What can I do here?

Head over to the 'Generate graphs' page and enter the username of a GitHub user. You can then select what kind of graph to generate based on that user.

If you'd like to see how everything works, follow the link to GitHub, where you can view everything that went into making this functional.

If you're interested in knowing more about this project, additional information can be found on the About page.
